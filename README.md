# redhat_1
REST service for basic CRUD operation on JSON dataset

Project Setup
=============

NPM
=====

To pull in all the required files for the client side:

```
npm install
```

####Run
To run project localy, go to main project folder and execute 'app.js' from command line.

```
node app.js
```

####Environments
*local
    *localhost:3001
